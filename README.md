#  IT-Systemelektroniker Professional Tool

A comprehensive Java-based system diagnostic and hardware testing tool designed for IT professionals. 

##  Features
* Full System Audit: Scans and displays PC Name, IP, CPU, RAM, and Storage directly from Windows WMI.
* Hardware Connected Devices: Lists all currently connected Plug & Play devices (USB, Drivers, etc.).
* Interactive Diagnostics Unit: 
  *  Mouse click tracking and verification.
  * Keyboard stroke detection.
  *  Full-screen dead-pixel color test.
  *  System speaker beep test.

##  Built With
* Java (Swing & AWT for GUI)
* Windows WMI (Windows Management Instrumentation) integration via ProcessBuilder.
